# Password Generator

A beautiful, simple, and secure password generator web application.

## Features

- **Customizable Length**: Choose password length between 4 and 64 characters.
- **Character Options**: Include/exclude Uppercase, Lowercase, Numbers, and Symbols.
- **One-Click Copy**: Easily copy the generated password to your clipboard.
- **Modern UI**: Clean, dark-themed interface with glassmorphism effects.
- **Responsive**: Works great on desktop and mobile.

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Variables, Animations)
- JavaScript (ES6+)
- Font Awesome (Icons)
- Google Fonts (Outfit)

## How to Use

1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Adjust the settings to your preference.
4. Click "Generate Password".
5. Click the copy icon to copy the password to your clipboard.
